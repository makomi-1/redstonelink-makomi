# 序号外显模式与近外显中心显示

生成时间：2026-03-16 23:50:02
文件名：2026-03-16_235002_序号外显模式与近外显中心显示.md

## 任务背景
用户要求在现有远外显基础上增加近外显中心显示，并把按键切换改为四态状态机；同时新增字体大小可配置能力。

## 方案详情

### 现状分析
- 当前仅有远外显开关（布尔），按键只能开/关切换。
- 当前无 HUD 层近外显渲染。
- 当前字体大小固定，不可配置。

### 技术方案
- 客户端配置改为四态模式：`far -> near -> both -> off`，并新增字体缩放配置 `client.serialOverlayFontScale`。
- 新增 HUD 渲染器：近距离（8 格内）准星命中可配对节点时，在屏幕中心绘制深色背景序号文本。
- 保留远外显颜色差异并复用到近外显。
- 继续使用同一个切换按键，不新增按键。
- 更新中英文切换提示文案与使用说明。
