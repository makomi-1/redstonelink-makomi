# 固定本地启动窗口为1920x1080

生成时间：2026-03-22 10:32:38
文件名：2026-03-22_103238_固定本地启动窗口为1920x1080.md

## 任务背景
本地开发环境每次启动客户端时，窗口尺寸不固定，当前运行目录仍保持全屏状态。需要将本地启动窗口固定为 1920x1080，并明确关闭全屏，避免每次手动调整。

## 方案详情

### 现状分析
- `build.gradle` 当前仅配置了基础的 Loom 模组信息，未为 `client` 运行配置声明启动宽高参数。
- `run/options.txt` 当前包含 `fullscreen:true`，且 `overrideWidth`、`overrideHeight` 均为 `0`，不能保证每次以窗口化 1920x1080 启动。

### 技术方案
- 在 `build.gradle` 的 `loom` 配置下补充 `runs.client.programArgs`，显式传入 `--width 1920 --height 1080`。
- 在 `run/options.txt` 中将 `fullscreen` 调整为 `false`，保持窗口化启动。

### 影响范围
- `build.gradle`
- `run/options.txt`

- **操作对象**：`run/options.txt`

- **预期结果**：本地客户端以窗口模式启动，而不是直接进入全屏
- **关键里程碑**：是
