# 增强文档化工作流上下文命中

生成时间：2026-03-10 21:42:17
文件名：2026-03-10_214217_增强文档化工作流上下文命中.md

## 任务背景
当前文档化工作流分散在 AGENTS.md 与 skills/docs-dev-docflow/SKILL.md 中，触发条件与必需信息缺少统一的“上下文命中速查规则”，导致执行时容易漏更或不确定要索要的信息。需要在不新增新流程的前提下，使现有工作流更容易被遵守和执行。

## 方案详情

### 现状分析
- AGENTS.md 仅引用文档化工作流，但缺少“触发事件 → 目标文档 → 必问信息”的速查规则。
- SKILL.md 已有文档级触发与模板，但缺少统一事件映射与最小提问清单。
- 当前目标仅限调整 AGENTS.md 与 SKILL.md，不涉及 docs_dev 其他文件。

### 技术方案
- 在 AGENTS.md 的“文档化自动工作流”下新增“上下文命中速查卡”，明确触发事件、目标文档与必问信息，并强调仅依据聊天记录、仅更新标注自动更新的文档、不创建缺失文档。
- 在 SKILL.md 中补充“上下文命中规则（事件→文档）”“最小提问清单”“优先级说明（与 AGENTS 冲突以 AGENTS 为准）”。

### 影响范围
- AGENTS.md
- skills/docs-dev-docflow/SKILL.md

## 原子步骤清单

### 步骤 1：新增上下文命中速查卡
- **操作对象**：AGENTS.md
- **具体动作**：在“文档化自动工作流”下新增上下文命中速查卡，明确触发事件、目标文档与必问信息；补充执行限制（仅依据聊天记录、仅更新标注自动更新的文档、不创建缺失文档）。
- **预期结果**：执行入口清晰、触发判定明确。
- **关键里程碑**：是

### 步骤 2：补充上下文命中规则与最小提问清单
- **操作对象**：skills/docs-dev-docflow/SKILL.md
- **具体动作**：补充事件→文档映射、最小提问清单与与 AGENTS 的优先级说明。
- **预期结果**：执行时无需在多段文档中来回查找，降低漏更。
- **关键里程碑**：是

## 预期结果
- 文档化工作流命中与执行更稳定。
- 触发条件与必问信息清晰可查，减少遗漏与反复确认。
