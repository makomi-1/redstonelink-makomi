# 显示结构化不展开

生成时间：2026-03-13 19:14:43
文件名：2026-03-13_191443_显示结构化不展开.md

## 任务背景
用户要求 GUI 当前连接、GUI 悬停 tooltip、物品悬停 tooltip 的序号展示改为“保持输入结构风格”，不再逐个展开大列表。

## 方案详情

### 现状分析
- `set_links` 输入已统一为 `N` / `A:B` + `/` 分隔。
- GUI 与 tooltip 当前仍使用逐项拼接展示，长列表会展开后再截断。
- 物品 tooltip 也基于逐项拼接，显示风格与输入结构不一致。

### 技术方案
- 新增通用“结构化展示”工具：将序号集合压缩为 `N` / `A:B`，再用 `/` 拼接。
- 在展示侧统一复用该工具，保留长度限制与 `(+n)` 提示，但不再逐项展开。
- 不改持久化结构，不新增存档字段，避免扩大改动面。
