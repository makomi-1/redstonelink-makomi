# 近外显当前连接保密改造

生成时间：2026-03-17 14:25:01  
文件名：2026-03-17_142501_近外显当前连接保密改造.md

## 任务背景
当前近外显第三行“当前连接”来自 `PairableNodeBlockEntity` 的公共同步字段，附近客户端可直接收到明文，无法按玩家权限控制。  
目标是在不改变第一/第二行显示语义的前提下，实现三种保密模式并支持“部分加密名单”运维命令。

## 方案详情

### 现状分析
- 方块实体更新包包含 `LinkedTargetSerials`，客户端 HUD 直接读取该快照。
- 网络层目前仅有配对界面 S2C 包，不含近外显 C2S 请求通道。
- 命令体系已有批量解析与 `confirm` 后缀能力，可复用到隐私名单维护。

### 技术方案
1. 配置扩展  
   - 新增 `server.currentLinksPrivacy.mode=hidden|masked|plain`。  
   - 新增 `server.currentLinksPrivacy.viewPermissionLevel=0~4`。

2. 新增服务端名单持久化  
   - 新建 `CurrentLinksPrivacySavedData`，按 `type + serial` 管理加密名单。  
   - 提供 add/remove/replace/list/contains，并支持退役联动清理。

3. 新增隐私判定服务  
   - 新建 `CurrentLinksPrivacyService`，集中实现 `hidden/masked/plain` 判定与权限等级检查。

4. 命令子包  
   - 新增 `/redstonelink privacy current_links mask ...`。  
   - 子命令：`add/remove/set/list`。  
   - `set` 复用批量写法与 `confirm` 二次确认。

5. 网络改造  
   - 在 `PairingNetwork` 新增 C2S 请求与 S2C 定向响应。  
   - 客户端请求“当前准星命中节点”的第三行数据；服务端按权限返回目标序号列表或空列表。  
   - 服务端校验请求距离与节点有效性，避免越权探测。

6. 客户端近外显改造  
   - `LinkSerialHudOverlayRenderer` 第三行改为读取网络缓存。  
   - 请求节流，命中变化时按需刷新；无数据或无权限统一显示 `-`。

7. 移除公共同步明文  
   - `PairableNodeBlockEntity` 去除 `LinkedTargetSerials` 更新包字段，杜绝被动泄露。

### 影响范围
- 服务端配置：`RedstoneLinkConfig`
- 服务端数据：`CurrentLinksPrivacySavedData`、`LinkRetireCoordinator`
- 服务端业务：`CurrentLinksPrivacyService`
- 命令：`ModCommands` + 新增 `command/privacy/*`
- 网络：`PairingNetwork`
- 客户端：`RedstoneLinkClient`、`LinkSerialHudOverlayRenderer`
- 方块实体同步：`PairableNodeBlockEntity`

- **操作对象**：`RedstoneLinkConfig`
- **具体动作**：新增 mode/permissionLevel 字段、解析、默认模板与 getter。
- **预期结果**：服务端可切换 `hidden/masked/plain` 策略。
- **关键里程碑**：是

- **预期结果**：保密策略具备可持久化与可复用判定入口。
- **关键里程碑**：是

### 步骤 3：命令子包落地
- **操作对象**：`command/privacy/CurrentLinksPrivacyCommandRegistry`、`ModCommands`
- **具体动作**：接入 add/remove/set/list，set 强制 confirm 与批量解析。
- **预期结果**：管理员可运维加密名单。
- **关键里程碑**：是

### 步骤 4：网络与客户端接入
- **操作对象**：`PairingNetwork`、`RedstoneLinkClient`、`LinkSerialHudOverlayRenderer`
- **具体动作**：新增 C2S 请求与 S2C 定向响应，近外显第三行改读缓存并节流请求。
- **预期结果**：无权限客户端无法得到明文连接。
- **关键里程碑**：是

### 步骤 5：移除方块实体公共同步明文
- **操作对象**：`PairableNodeBlockEntity`
- **具体动作**：删除 `LinkedTargetSerials` 更新包读写路径。
- **预期结果**：从同步源头消除明文广播。

- 近外显第三行实现按权限可见，满足 `全保密/部分加密/全解密` 三模式。
