# 命令读取隐私并入

生成时间：2026-03-17 20:45:35
文件名：2026-03-17_204535_命令读取隐私并入.md

## 任务背景
- 当前 `node get` 与 `link get` 读取链路直接输出 `LinkSavedData` 原始目标集合，未并入现有“当前连接”隐私控制。
- GUI、近外显、物品栏快照已接入 `CurrentLinksPrivacyService`，命令读取存在策略缺口。

## 方案详情

### 现状分析
- `ModCommands.executeNodeGet/executeLinkGet` 直接读取并格式化目标集合。
- `CurrentLinksPrivacyService` 已提供来源判定与目标级过滤能力，但入口主要面向玩家与物品快照。

### 技术方案
- 在 `CurrentLinksPrivacyService` 新增“命令读取入口”：按 `ServerLevel + hasViewPermission` 解析可见当前连接集合。
- `ModCommands.executeNodeGet/executeLinkGet` 统一改为调用该入口。
- 命令权限使用 `source.hasPermission(server.currentLinksPrivacy.viewPermissionLevel)`。
- 使用说明补充 `node get/link get` 受隐私模式控制。

### 影响范围
- `src/main/java/com/makomi/data/CurrentLinksPrivacyService.java`
- `src/main/java/com/makomi/command/ModCommands.java`
- `docs_dev/使用说明.md`

## 语义对齐说明
- 术语保持 `triggerSource/core`，不新增旧术语输出。
- 方向映射保持 `triggerSource -> core`、`core -> triggerSource`。
- 未引入 `com.makomi.api.v1.*`。
- 兼容边界保持“读旧写新”，不新增存储键。
