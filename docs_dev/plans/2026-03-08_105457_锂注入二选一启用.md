# 锂注入二选一启用

生成时间：2026-03-08 10:54:57

## 任务背景
用户要求将当前“有锂时原版+锂并行启用”调整为“二选一”策略，避免同一环境同时启用两套注入。

## 方案详情

### 现状分析
- `redstonelink.mixins.json` 同时注册 `RedStoneWireBlockMixin` 与 `RedStoneWireBlockLithiumMixin`。
- `RedstoneLinkMixinPlugin` 仅对 lithium mixin 做条件控制，原版 mixin 无条件启用。

### 技术方案
- 在插件中新增 `VANILLA_MIXIN` 常量。
- `shouldApplyMixin` 统一读取 `hasLithium`：
  - 命中 `VANILLA_MIXIN` 时返回 `!hasLithium`。
  - 命中 `LITHIUM_MIXIN` 时返回 `hasLithium`。
  - 其他 mixin 保持 `true`。

### 影响范围
- `src/main/java/com/makomi/mixin/plugin/RedstoneLinkMixinPlugin.java`

## 原子步骤清单

- **具体动作**：新增原版 mixin 常量并调整 `shouldApplyMixin` 分支
- **预期结果**：有锂仅启用锂注入；无锂仅启用原版注入
