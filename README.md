# RedstoneLink

## 项目说明
这是一个基于 Minecraft Fabric 1.21.1 的红石联动模组项目，核心目标是通过“连接核心 + 连接按钮”的配对机制，把分散的红石触发行为组织成可管理、可扩展的联动网络。

当前版本已完成基础注册、配方与配对链路，支持多类核心/按钮方块的配对触发，并持续围绕红石粉核心附着、触发模式与稳定性进行迭代。

## 当前状态
- 开发状态：持续迭代中（非最终发布版）。
- 测试状态：已进行基础构建与简单功能验证。
- 适用环境：Minecraft 1.21.1 + Fabric Loader（详见 `fabric.mod.json` 与构建配置）。
